## üöÄ Features
This is a fully functional social media application built with Django.
* **User Authentication:** Secure Login, Registration, and Logout.
* **Profile System:** Users have profiles with stats (Followers, Following, Posts).
* **Feed:** Dynamic feed showing posts from followed users.
* **Create Posts:** Upload text and images.
* **Interactions:** Like and Comment system.
* **Direct Messaging:** Real-time chat with auto-clearing input.
* **Search:** Search functionality to find other users.
* **Account Management:** Users can delete their own posts and account.

## ‚öôÔ∏è How to Run Locally
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/S4544/social-media-platform.git](https://github.com/S4544/social-media-platform.git)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd social-media-platform
    ```

3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
4.  **Run Migrations:**
    ```bash
    python manage.py migrate
    ```
5.  **Start the Server:**
    ```bash
    python manage.py runserver
    ```
